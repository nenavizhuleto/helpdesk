# API Documentation

## Routes

All API routes placed under `http://host:port/api/*` path 

## Versioning

API's versions separeted with `/vN` section

Example:

> GET API version

```
curl http://localhost:3000/api/v1
```

Output:

```json
{
    "version": "1.0"
}
```
